# Project Description

Plenum is a responsive and interactive e-commerce website designed to showcase and sell furniture products. Users can browse through various furniture categories, view product details, and interact with a dynamic user interface built with HTML, CSS, and JavaScript. This project demonstrates front-end development skills, UI/UX design, and dynamic client-side functionality.

# Features

Home Page: Highlights featured furniture products with a clean, modern layout.

Product Listing: View furniture items by categories such as sofas, chairs, tables, and more.

Product Details: Detailed view for each product with images, price, and description.

Responsive Design: Fully responsive layout for desktop, tablet, and mobile devices.

Interactive UI: Dynamic product hover effects, smooth navigation, and modal pop-ups.

Search & Filter (Optional JS Implementation): Users can search for specific products or filter by categories.

Shopping Cart (Optional JS Implementation): Add products to a cart and calculate totals.
